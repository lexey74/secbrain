# 1. Роль и Контекст
Мы разрабатываем Telegram-бота "Second Brain".
**Цель:** Сбор ссылок/мыслей, транскрибация через Ollama, сохранение в Obsidian.

# 2. Технический стек (Строго)
* **Язык:** Python 3.12
* **Telegram:** pyTelegramBotAPI (telebot)
* **AI:** Ollama (через requests/API)
* **База знаний:** Obsidian (работа с Markdown файлами)

# 3. Правила поведения (Answering Rules)
1.  **Язык:** Внутренние размышления на английском, **ответ пользователю всегда на РУССКОМ**.
2.  **Код:**
    * Всегда используй Type Hinting.
    * Используй модуль \`logging\` вместо \`print\`.
    * Не извиняйся и не представляйся. Сразу к делу.
3.  **Безопасность:** Не хардкодь токены, используй \`os.getenv\`.

# 4. Процесс
Если нужно создать файл — **создавай его**. Не спрашивай разрешения, если это не удаляет данные.
