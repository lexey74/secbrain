# Инструкция по развертыванию (Docker)

## Предварительные требования
- Docker и Docker Compose установлены.
- Ollama запущена на хосте (или доступна по сети).
- Telegram Bot Token получен.

## Шаг 1: Настройка окружения
1. Скопируйте пример конфига:
   ```bash
   cp .env.example .env
   ```
2. Отредактируйте `.env`:
   - Вставьте `TELEGRAM_BOT_TOKEN`.
   - Укажите ваше имя пользователя в `USER_NAME` (это название папки в `users/`, например `lexey`).

## Шаг 2: Структура папок
Убедитесь, что папка пользователя существует (она должна была создаться автоматически, но на всякий случай):
```bash
mkdir -p users/lexey/downloads
mkdir -p temp
mkdir -p bot_state
```

## Шаг 3: Запуск
Запустите бота в фоновом режиме:
```bash
docker compose up -d --build
```
Флаг `--build` пересобирает контейнер, если были изменения в коде.

## Шаг 4: Проверка
Посмотрите логи, чтобы убедиться, что всё работает:
```bash
docker compose logs -f
```

## Обновление
Чтобы обновить бота (после `git pull`):
```bash
docker compose up -d --build
```

## Полезные команды
- Остановить: `docker compose down`
- Перезагрузить: `docker compose restart`
